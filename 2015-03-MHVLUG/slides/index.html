0. Getting Started 


 curl https://install.meteor.com/ | sh
Installs Meteor libraries and tools, Node and Mongo dependencies. (Windows support pending)

1. Create Boilerplate App 


meteor create webapp
This creates the basic scaffolding and code (HTML views, JS controller, CSS style). 

webapp.js     // isomorphic codebase, loads at client and server
webapp.html   // contains body, head and templates
webapp.css    // app styles
.meteor       // meteor configuration, local db
Note that directory structure and file names can impact how application is built and bundled for deployment.

2. Run/Deploy Your App 


cd webapp
meteor run
This runs the Meteor server on localhost (default port 3000).
If a remote Mongo server is not set, this also starts a local Mongo Daemon (database 'meteor'). 


meteor deploy webapphv.meteor.com  (-- delete)
For production deployment on AWS, Digital Ocean, Joyent or other cloud providers
use the Meteor Up (mup) tool. The 'deploy' option is for validation only.
